# Conditional Diffusion Models for ESM Precipitation

A Reveal.js presentation on **Aich et al.** — conditional diffusion models for downscaling and bias correction of Earth system model precipitation.

---

## Topics

- Shared embedding space methodology (QDM + noise)
- Forward & reverse diffusion process
- U-Net architecture & conditioning
- Main results (South America climate signal)
- Literature connections (ScoreGrad, Chen, Gao, SBTS)
